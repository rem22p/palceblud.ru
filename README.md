# 🎹 пальцеблуд.рф

**Современный тренажёр слепой печати на React 19 + Vite 6**

---

## 🚀 Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build
```

## 🎯 Режимы работы

### Practice Mode
- Свободная печать на скорость
- Таймер: 15s / 30s / 60s / 120s
- Статистика: WPM, Accuracy, Raw, Consistency

### Learning Mode
- 6 уроков с прогрессией
- От домашней строки до скоростной печати
- Трекер прогресса и достижения

## 🎨 Настройки темы

Тема настраивается в `styles/theme.css` через CSS переменные:

```css
:root {
  --background: #1a1a1a;
  --foreground: #d1d0c5;
  --accent: #e2b714;  /* Золотой акцент */
}
```
